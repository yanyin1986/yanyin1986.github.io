## 分支管理
平时在做完merge之后，就会涉及到删除不需要的branch。
那么怎么去做呢？

1. 查看历史记录里的 `Merge pull request`的合并的PR记录
2. 然后去一个个删除不需要的分支

## 查看分支
```bash
git log --oneline --decorate | grep 'Merge pull request' -m 10
```

## 删除分支
```bash
git branch -d ${branch_name}
```
